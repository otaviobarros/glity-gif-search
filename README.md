## Prerequisites
* npm or yarn

## Installation

1. Clone the repo

```bash
git clone https://github.com/otaviobarros/glity-gif-search
```
2. Install all packages
```bash
npm install
# or
yarn
```
3. Start the server
```bash
npm run dev
# or
yarn dev
```
